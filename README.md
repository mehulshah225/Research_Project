# EXORCISM-5: Containment-Based Postprocessing for Quantum-Cost-Aware ESOP Synthesis

This repository implements EXORCISM-5, a containment-driven postprocessing framework for optimizing Exclusive-Sum-of-Products (ESOP) circuits generated by EXORCISM-4.

EXORCISM-5 reduces quantum realization cost by detecting cube containment structures and transforming ESOP representations into optimized Product-Sum / EOSOPS forms, leading to lower quantum cost and T-count.

---

## Overview

The synthesis pipeline performs the following steps:

1. Read Boolean benchmark functions in **PLA format**
2. Run **EXORCISM-4** to generate ESOP expressions
3. Apply **EXORCISM-5 postprocessing**
   - cube containment detection
   - cube merging
   - PS transformation
   - EOSOPS generation
4. Apply Final-stage refinement
   - additional structural simplification
   - post-mapping cleanup of EOSOPS
5. Evaluate quantum cost using:
   - Maslov cost model
   - T-count (Clifford+T decomposition proxy)
6. Export results to Excel (multi-sheet report)

## Benchmark Categories

The repository includes three benchmark suites.

### 1. EPFL-style / Small ESOP Benchmarks
Small and structured ESOP functions used to evaluate containment-based optimizations.

Examples:
- sym_d
- esops1
- life_d_75
- max46_d

---

### 2. MCNC / RevLib Benchmarks
Standard reversible logic synthesis benchmarks.

Examples:
- rd53f1
- rd84f4
- sao2f3
- t481_d
- xor8

---

### 3. Large-Scale Synthetic Benchmarks
Synthetic stress-test benchmarks with 100 input variables.

Examples:
- f_100_400-*
- f_100_600-*

These are used to evaluate scalability and large-scale optimization performance.

## Core Algorithms

EXORCISM-5 introduces several optimization stages.

### Cube Containment Detection
Detects whether one cube is fully or partially contained within another.

### Cube Merging
Merges compatible cubes while preserving Boolean equivalence.

### Product-Sum Transformation
Transforms containment structures into Product-Sum expressions.

### Final-stage Refinement
Performs post-mapping simplification for additional quantum cost reduction.

---

## Quantum Cost Model

Maslov Cost
Models decomposition cost of multi-controlled Toffoli gates in reversible circuits.

T-count
Estimates fault-tolerant cost under Clifford+T decomposition:
   - NOT / CNOT → 0 T-cost
   - Multi-controlled gates → linear approximation model
---

## Key Experimental Results

- Up to **70.6% reduction** in Maslov cost (structured benchmarks)
- Approximately **66–67% average reduction** on large synthetic benchmarks
- Approximately **59–60% reduction** in T-count
- No benchmark exhibited quantum cost increase after postprocessing

# EXORCISM-5: Containment-Based Postprocessing for Quantum-Cost-Aware ESOP Synthesis

This repository contains the implementation and benchmark suite for **EXORCISM-5**, a containment-based postprocessing framework for optimizing Exclusive-Sum-of-Products (ESOP) representations generated by EXORCISM-4.

EXORCISM-5 reduces quantum realization cost by identifying containment relationships among ESOP cubes and transforming them into optimized Product-Sum (PS) / EOSOPS representations.

---

## Overview

The synthesis pipeline performs the following steps:

1. Read Boolean benchmark functions in **PLA format**
2. Run **EXORCISM-4** to generate ESOP expressions
3. Apply **EXORCISM-5 postprocessing**
   - cube containment detection
   - cube merging
   - PS transformation
   - EOSOPS generation
4. Compute quantum realization cost using the **Maslov cost model**
5. Generate an Excel summary report

## Benchmark Categories

The repository includes three benchmark suites.

### 1. EPFL-style / Small ESOP Benchmarks
Small and structured ESOP functions used to evaluate containment-based optimizations.

Examples:
- sym_d
- esops1
- life_d_75
- max46_d

---

### 2. MCNC / RevLib Benchmarks
Standard reversible logic synthesis benchmarks.

Examples:
- rd53f1
- rd84f4
- sao2f3
- t481_d
- xor8

---

### 3. Large-Scale Synthetic Benchmarks
Synthetic stress-test benchmarks with 100 input variables.

Examples:
- f_100_400-*
- f_100_600-*

These are used to evaluate scalability and large-scale optimization performance.

Run the complete benchmark pipeline:

```bash

```

The script automatically:

- finds all `.pla` files in `benchmarks/`
- runs EXORCISM-4
- applies EXORCISM-5
- computes Maslov cost
- exports benchmark statistics

## Core Algorithms

EXORCISM-5 introduces several optimization stages.

### Cube Containment Detection
Detects whether one cube is fully or partially contained within another.

### Cube Merging
Merges compatible cubes while preserving Boolean equivalence.

### Product-Sum Transformation
Transforms containment structures into Product-Sum expressions.

### EOSOPS Synthesis
Produces optimized Exclusive-Or Sum-of-Product-Sums representations.

---

## Quantum Cost Metric

Circuit cost is evaluated using the **Maslov quantum cost model**, which estimates the realization cost of multi-controlled Toffoli gates in reversible and quantum circuits.

This metric is used throughout the accompanying paper for quantitative evaluation.

---

## Key Experimental Results

- Up to **70.58% reduction** in Maslov cost on structured benchmarks
- Approximately **66–67% average reduction** on large synthetic benchmarks
- No benchmark exhibited quantum cost increase after postprocessing
